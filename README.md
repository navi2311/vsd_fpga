# vsd_fpga
This repo contains information related to fpga fabric workshop by vsdiat

In this workshop we run two designs(counter,riscv_core) in following tools and analysis the power,timing,ultilization,post-implementation netlist.
- Vivado
- openfpga(VTR--which has VPR framework)
- SOFA(which using open fpga and maps the technology to sky130 pdk)
