# vsd_fpga
This repo contains information related to fpga fabric workshop by vsdiat

In this workshop we run two designs`(counter,riscv_core)` in following tools  
- Vivado
  - implement custom logic and functional simuation
  - intergate vivado ip in custom logic
  - how to use ILA to view waveforms
- openfpga(VTR--which has VPR framework)
  - how to run VPR flow
  - how to run VTP flow
- SOFA(which uses open fpga and maps the technology to sky130 pdk)
  - how to run SOFA flow

in these tools analysed the power,timing,ultilization,post-implementation netlist verfication for both designs
